# RPG Party Analyzer

## Project Overview

RPG Party Analyzer is a Java application that evaluates how effectively a group of RPG characters will perform together. The application analyzes character statistics, roles, abilities, and class relationships to measure the party's overall balance and synergy.

---

## Intended User

This application is designed for gamers and other users who enjoy creating and evaluating RPG party compositions.

---

## Version 1

Version 1 will allow users to enter character information, including:

- Character Name
- Character Class
- Party Role
- Attack
- Defense
- Health Points (HP)
- Special Abilities

The application will analyze the completed party and provide ratings for:

- Attack
- Defense
- Survivability
- Ability Coverage
- Class Synergy

These categories will contribute to an overall party percentage, letter grade, and x/5 rating.

---

## Future Goals

Future versions may include:

- Automatic party builder
- Boss encounter analysis
- Equipment and item support
- Additional RPG archetypes
- Expanded synergy calculations
